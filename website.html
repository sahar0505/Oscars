THE OSCARS

Mon projet explore l'évolution des nominations et des victoires aux Oscars depuis leur création jusqu'à 2022. En mettant en lumière les divers genres de films nominés et les brillants réalisateurs derrière ces œuvres, j'offre un aperçu des succès cinématographiques à travers les décennies.

Word Cloud

Utilisant Python pour generer un nuage de mots ("word cloud" en anglais) à partir des noms des réalisateurs de films nominés aux Oscars. Le nuage de mots est une représentation visuelle où la taille des mots est proportionnelle à leur fréquence d'apparition dans le texte source. En utilisant les noms des réalisateurs de films, le code crée une image où les noms des réalisateurs les plus fréquents apparaissent en plus gros, formant ainsi une représentation graphique mettant en évidence les réalisateurs les plus prolifiques ou les plus souvent associés à des films nominés aux Oscars. Cela permet une visualisation rapide des réalisateurs les plus influents ou les plus présents dans le contexte des nominations aux Oscars.

Bar representation

utilisant la bibliothèque Plotly Express pour créer ce graphique interactif de type barre. j'ai pris en compte les données des genres de films nominés et gagnants aux Oscars pour la catégorie "Meilleur Film". Le graphique interactif permet aux utilisateurs de survoler les barres pour voir les chiffres exacts, ainsi que de sélectionner ou désélectionner les catégories dans la légende pour une visualisation plus précise des données.

Sunburst representation

utilisant toujours la meme bibliothèque Plotly Express pour generer ce graphique interactif de type sunburst. Le sunburst est une représentation visuelle hiérarchique qui affiche les données sous forme de secteurs concentriques. Le graphique interactif Sunburst généré ici montre la répartition des réalisateurs de films en fonction des genres sur différents niveaux hiérarchiques. Chaque secteur représente un réalisateur, et à l'intérieur de chaque secteur, les genres de films avec lesquels ce réalisateur est associé sont affichés.

Download

Pour ce projet, j'ai utilisé un dataset provenant de Kaggle contenant les gagnants et les nommés aux Oscars pour la catégorie du Meilleur Film, accompagné d'évaluations d'IMDB et de Rotten Tomatoes, ainsi que d'autres détails comme la durée des films et les informations sur les réalisateurs. Deux fichiers sont disponibles en téléchargement : le fichier CSV des Oscars et le fichier Python contenant les scripts pour les plots interactifs Ce fichier Python offre la possibilité de modifier les scripts pour créer différents types de visualisations ou explorer d'autres aspects des données selon les besoins individuels ou les intérêts spécifiques
